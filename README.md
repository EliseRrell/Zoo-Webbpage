This website presents a fictional zoo with information about animals, opening hours, ticket prices, visitor guidelines and a newsletter subscription form.
It is designed to be lightweight, fast and fully responsive without relying on JavaScript. The webbpage is in Swedish.

Features
Sticky navigation header with logo and quick links
CSS‑only image slider showcasing the zoo’s animals
Responsive layout optimized for desktop and mobile
Opening hours & ticket price tables
Visitor information section with embedded map
Newsletter form with validation
